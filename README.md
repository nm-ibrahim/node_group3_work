>Project Description
This project is a backend REST API built with Node.js and Express.  
It is used to manage employee data such as name, email, and age.  
The API allows users to create, read, update, and delete employee records stored in a MySQL database.

This project was developed as part of a group assignment to practice backend development, database integration, and API creation.


>Technologies Used
- Node.js
- Express.js
- MySQL 
- npm
- Git and github

>Database Setup Steps
1. Start your MySQL server.
2. Create a new database:
    CREAT DATABASE node_group3_work
